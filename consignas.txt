Plataforma Inteligente de Compra y 
Venta de Autos Usados 
1. Introducción 
El mercado de autos usados ha crecido significativamente en los últimos años, impulsado 
por plataformas digitales que permiten conectar compradores y vendedores de manera 
directa. Sin embargo, muchas veces los compradores enfrentan dificultades para evaluar el 
estado real de un vehículo o determinar si el precio solicitado es razonable. 
El objetivo de este proyecto es desarrollar una plataforma web de compra y venta de 
autos usados, donde los vendedores puedan publicar vehículos y los compradores puedan 
buscarlos, analizarlos y compararlos. 
Como elemento innovador, el sistema deberá integrar inteligencia artificial para analizar 
fotografías del vehículo, permitiendo estimar: 
● el estado general del automóvil 
● posibles daños visibles 
● una estimación aproximada del valor de mercado 
Este proyecto permitirá aplicar conceptos fundamentales de desarrollo web como: 
● arquitectura cliente-servidor 
● desarrollo de interfaces web dinámicas 
● APIs backend 
● manejo de imágenes 
● integración con servicios de inteligencia artificial 
● despliegue de aplicaciones web 
2. Objetivos del Proyecto 
El objetivo general del proyecto es desarrollar una plataforma web completa que permita 
publicar, buscar y analizar autos usados. 
Objetivos específicos: 
● Desarrollar una interfaz web para publicar y buscar vehículos. 
● Implementar un backend que gestione usuarios y publicaciones. 
● Integrar inteligencia artificial para analizar imágenes de vehículos. 
● Facilitar la comparación de vehículos para compradores. 
● Implementar filtros de búsqueda avanzados. 
● Publicar el sistema en un hosting gratuito accesible públicamente.

3. Descripción del Sistema 
La plataforma deberá permitir la interacción entre dos tipos de usuarios principales. 
1. Vendedor 
Es el usuario que desea publicar un vehículo en la plataforma para venderlo. 
Podrá: 
● registrarse en la plataforma 
● iniciar sesión 
● publicar autos usados 
● subir imágenes del vehículo 
● editar publicaciones 
● eliminar publicaciones 
● recibir consultas de compradores 
2. Comprador 
Es el usuario que desea buscar autos disponibles en la plataforma. 
Podrá: 
● registrarse 
● buscar autos disponibles 
● aplicar filtros de búsqueda 
● ver el detalle de cada vehículo 
● analizar la estimación de estado generada por IA 
● comparar diferentes vehículos 
● contactar al vendedor

4.2 Publicación de vehículos 
Los vendedores podrán publicar autos proporcionando información como: 
● marca 
● modelo 
● año 
● kilometraje 
● tipo de combustible 
● transmisión 
● precio 
● ubicación 
● descripción del vehículo 
Además, deberán poder subir múltiples fotografías del automóvil. 
4.3 Búsqueda de vehículos 
Los compradores podrán buscar autos mediante: 
● marca 
● modelo 
● rango de precio 
● año 
● kilometraje 
● ubicación 
Los resultados deberán mostrarse en forma de listado con información resumida. 
4.4 Visualización de detalles 
Cada vehículo deberá tener una página de detalle que incluya: 
● galería de imágenes 
● características del vehículo 
● descripción 
● precio 
● ubicación 
● análisis realizado por inteligencia artificial

4.5 Inteligencia artificial para análisis de 
imágenes 
El sistema deberá integrar un servicio de inteligencia artificial capaz de analizar las 
fotografías del vehículo. 
La IA deberá intentar: 
● identificar daños visibles 
● estimar el estado general del auto 
● sugerir un rango estimado de precio 
La estimación podrá mostrarse como: 
● excelente estado 
● buen estado 
● estado regular 
● requiere reparación 
4.6 Comparación de vehículos 
Los compradores podrán seleccionar varios vehículos y compararlos en base a: 
● precio 
● año 
● kilometraje 
● características principales 
4.7 Contacto entre comprador y 
vendedor 
El sistema deberá permitir que un comprador envíe una consulta al vendedor sobre un 
vehículo.
